# SilverStripe Appointment Module

## Requirements

* SilverStripe CMS 4.0+
* SilverStripe Assets 1.0+
* SilverStripe Asset Admin Module 1.0+
* wwnorden/vehicles

## Installation

```
composer require wwnorden/appointments
```

## Documentation
 
Create an appointment and add vehicles to them.
